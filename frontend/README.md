# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/f58b81d9-0309-4e96-afa3-55dea456a21a

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy [.env.example](.env.example) to `.env.local` and set `VITE_API_BASE_URL` to your QuizBot Studio backend (default `http://127.0.0.1:8000`)
3. Run the app:
   `npm run dev`
